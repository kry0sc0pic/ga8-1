EPS Calculation (Sample Code)


import pandas as pd

# Quarterly income statement snapshot
df = pd.DataFrame({
    "net_income": [510_000_000, 540_000_000],
    "shares_outstanding": [150_000_000, 156_000_000]
}, index=["Q3", "Q4"])

def compute_eps(row):
    return row["net_income"] / row["shares_outstanding"]

df["eps"] = df.apply(compute_eps, axis=1)

print(df[["eps"]])
Walk through how EPS is computed programmatically. Emphasize that this is illustrative and that production calculations must align with accounting policies and disclosures.

Key Financial Formulas

Return on Equity (ROE): \[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

Net Interest Margin (NIM): \[ \text{NIM} = \frac{\text{Interest Income} - \text{Interest Expense}}{\text{Average Earning Assets}} \]

Cost-to-Income Ratio: \[ \text{C/I} = \frac{\text{Operating Expenses}}{\text{Operating Income}} \]

Keep the audience aligned on the core metrics. Explain the intuition behind each formula, especially ROE and NIM, as they are central to the bank's performance narrative.

Capital Adequacy & Risk-Weighted Assets

Capital Adequacy Ratio (CAR): \[ \text{CAR} = \frac{\text{Tier 1 Capital} + \text{Tier 2 Capital}}{\text{Risk-Weighted Assets}} \]
